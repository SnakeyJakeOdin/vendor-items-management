Reverse every word in an input string

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Write a Java program that reads a string from the input, reverse every word in the string and prints the reversed words in separate lines. Respect the coding style conventions of Java.
Sample Input:
Hello World

Sample Output:
olleH

dlroW


Sample Input:
Java is cool

Sample Output:
avaJ

si

looc



Memory limit: 256 MB
Time limit: 5 seconds
