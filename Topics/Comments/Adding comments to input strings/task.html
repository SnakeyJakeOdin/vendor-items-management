Adding comments to input strings

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
In Java, teachers often suggest beginners to use comments in their codes to increase readability and understandability. Write a simple Java program that accepts a string from a user as input and returns the same string, but commented. Remember, comments in Java can be either single-line (//) or multi-line (/* */), choose any that makes the string a comment.
Sample Input:
Hello, World!

Sample Output:
// Hello, World!


Sample Input:
This is a test string.

Sample Output:
// This is a test string.



Memory limit: 256 MB
Time limit: 5 seconds
